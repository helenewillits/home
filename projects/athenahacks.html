Athenahacks

MLH Hackathon





Athenahacks is a 24 hour hackathon hosted by MLH. For the hackathon, I used Java and Android Studio to design and code a user-friendly app that documents travel locations, participants, and event memos.

Obstacles

As a team, my partner and I overcame issues with data storage, graphics, and object referencing. While neither of us had used Java beyond high school AP Comp Sci, we utilized the mentors available at the hackathon and online resources to learn about proper code structure.

Additional Design Ideas

Beyond the features that we implemented, we conceptualized that the app would use a Google Mas API to allow travel photos to be mapped and we planned for the trip data to ultimately upload to and auto-populate a Shutterfly photobook.
